# Vietnam Urban Planning DB

A database project for urban planning data in Vietnam.

## Description

Vietnam Urban Planning DB is a structured database project that collects and organizes urban planning information across Vietnam. It aims to provide accessible data on city development, zoning regulations, and land use planning for research, analysis, and public reference.

## Initial Structure

```
vietnam-urban-planning-db/
├── README.md
└── .git/
```

## Purpose

This project serves as a centralized resource for:
- Urban planning regulations and policies
- Land use data
- City development information
- Zoning maps and boundaries
